### Пеньков Фёдор - инфраструктура, базовый CRUD и ядро RBAC
*   **Инфраструктура:** инициализация проекта, настройка окружения (Docker/Sail, .env), проектирование базовой схемы БД
*   **Система авторизации:** настройка сессионной аутентификации (Laravel Breeze)
*   **RBAC:** middleware для RBAC
*   **ACL:** проверки доступа через policy

### Смирнов Александр -  публичный интерфейс и взаимодействие пользователей
*   **Гостевой доступ:** настройка маршрутизации и UI для неавторизованных пользователей
*   **Система комментариев:** проектирование таблицы комментариев, разработка контроллера для добавления комментариев к постам
*   Страница детального просмотра поста
*   **Сид БД:** автоматическое наполнение БД

### Ларюшкин Евгений - модуль администрирования
*   Админка
*   **Управление пользователями:** реализация интерфейса для просмотра всех пользователей системы
*   **Управление правами:** Внедрение логики изменения ролей пользователей

### Муромский Андрей - модерация, файловая система и тестирование
*   **Процесс модерации:** разделение постов на черновики и опубликованне статьи, реализация пайплайна публикации
*   **Работа с файлами:** настройка загрузки изображений для постов
*   **Тестирование:** функциональные тесты

### Раскостов Павел - обработка ошибок и баг-трекинг
